## 3. 发现项与建议

### 3.1 需修复项

| 编号 | 问题 | 位置 | 说明 | 优先级 |
|------|------|------|------|--------|
| I1 | pgvector 索引创建SQL需验证 | store.py | DDL 需要实际 PostgreSQL 环境验证 | 低 |
| I2 | Embedding mock 需替换为真实 API | embedding_client.py | 当前使用 mock 向量，生产环境需替换 | 中 |
| I3 | Session cleanup 机制未实现过期扫描 | store.py | 当前仅支持主动 touch，缺少后台清理任务 | 低 |

### 3.2 改进建议

| 编号 | 建议 | 说明 | 优先级 |
|------|------|------|--------|
| S1 | 增加 Embedding 缓存持久化 | 重启后缓存丢失，建议支持磁盘缓存 | 低 |
| S2 | 增加 Memory 导出/导入接口 | 便于用户备份 | 低 |
| S3 | 增加知识库 Web 管理界面 | 当前仅支持文件导入 | 未来 |
